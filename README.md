# 🚀 Zero Gravity Web App

A fun interactive physics-based web app inspired by Google Antigravity.

## ✨ Features
- Floating objects with zero gravity effect
- Drag and throw (inertia)
- Mouse interaction (repel / attract)
- Object collision
- Particle effects
- Sound effects
- Wormhole teleportation
- Custom text objects
- Multiple color themes

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (Vanilla)
- Canvas API
- Web Audio API
